# IKEA Sales Dashboard – Power BI

 📊 Overview
This project presents an interactive Power BI dashboard designed to analyze IKEA sales data.  
The goal is to transform raw data into meaningful insights that support business decision-making.

---

 🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing products  
- Track revenue trends over time  
- Understand customer behavior and patterns  

---

  Tools & Technologies
- Power BI  
- Power Query  
- Excel (Data Source)  

---

📈 Key Features
- Interactive dashboard with filters and slicers  
- KPIs for sales, profit, and performance  
- Visualizations including bar charts, pie charts, and trend analysis  
- Data cleaning and transformation using Power Query  

---

 Insights
- Identified best-selling products and categories  
- Analyzed sales trends across different periods  
- Highlighted key factors affecting revenue
